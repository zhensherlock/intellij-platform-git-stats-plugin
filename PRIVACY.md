# Privacy Policy

Last updated: 2026-05-31

GitStats is an IntelliJ Platform plugin that displays Git activity statistics for the project opened in the IDE.

## Data processed locally

GitStats reads metadata from the current project's local Git history, including:

- Git author names
- Commit hashes and commit dates
- File paths reported by `git log --numstat`
- Added line counts, deleted line counts, modified file counts, and commit counts

GitStats does not read or transmit source file contents.

## No data collection or transmission

GitStats does not collect analytics, telemetry, crash reports, or usage statistics.

GitStats does not send source code, Git history, author names, file paths, repository paths, or generated statistics to the plugin author or to any external service.

## Local execution

GitStats invokes the Git executable configured in the IDE to calculate statistics. Results are shown only inside the IDE tool window.
